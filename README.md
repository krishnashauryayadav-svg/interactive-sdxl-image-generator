# 🚀 Interactive SDXL Image Generator with Google Drive Auto-Save

यह एक पायथन-आधारित इंटरैक्टिव एआई इमेज जनरेटर टूल है जो दुनिया के सबसे बेहतरीन ओपन-सोर्स इमेज मॉडल **SDXL 1.0 (Stable Diffusion XL)** का इस्तेमाल करता है। यह कोड पूरी तरह से मुफ्त वाले **Google Colab (T4 GPU)** के लिए ऑप्टिमाइज्ड है और इसमें जनरेट की गई तस्वीरें सुरक्षित रूप से आपके Google Drive में सेव होती हैं।

## ✨ खास फीचर्स (Features)
- **💬 Interactive Prompt Loop:** बार-बार कोड बदलने की ज़रूरत नहीं! कोलाब में ही इनपुट बॉक्स खुलेगा, प्रॉम्प्ट डालिए और एंटर दबाइए।
- **🎲 Smart Random Naming:** हर फोटो का नाम `ai_image_gen_{रैंडम_नंबर}.png` के रूप में सेव होगा, जिससे पुरानी फोटो डिलीट (overwrite) नहीं होगी।
- **☁️ Google Drive Auto-Save:** कोलाब रनटाइम क्रैश होने पर भी आपकी तस्वीरें हमेशा के लिए आपके ड्राइव में सुरक्षित रहेंगी।
- **🛡️ NSFW Filter Connected:** सुरक्षित उपयोग के लिए बैकएंड में नेगेटिव प्रॉम्प्ट्स का फिल्टर लगा हुआ है।
- **⚡ VRAM Optimized:** `pipe.enable_model_cpu_offload()` का इस्तेमाल किया गया है ताकि कम मेमोरी (Out of Memory Error) में भी कोड पानी की तरह चले।

## 🛠️ चलाने का पूरा तरीका (Step-by-Step Procedure)

### स्टेप 1: Google Colab सेटअप करें
1. [Google Colab](https://colab.research.google.com/) पर एक नया नोटबुक खोलें।
2. ऊपर मेनू में **Runtime** > **Change runtime type** पर जाएं और Hardware accelerator में **T4 GPU** सिलेक्ट करके सेव करें।

### स्टेप 2: Hugging Face टोकन प्राप्त करें (ज़रूरी)
चूंकि SDXL एक बड़ा मॉडल है, इसे डाउनलोड करने के लिए एक फ्री टोकन की ज़रूरत होती है:
1. [Hugging Face Settings](https://huggingface.co) पर जाएं।
2. **Create New Token** पर क्लिक करें, Type में **Read** चुनें और टोकन को कॉपी कर लें।

### स्टेप 3: लाइब्रेरीज़ इंस्टॉल करें
कोलाब के पहले सेल में नीचे दिया गया कमांड चलाकर रन करें:
```bash
!pip install -r requirements.txt
```
*(या सीधे चलाएं: `!pip install -q diffusers transformers accelerate torch huggingface_hub`)*

### स्टेप 4: Hugging Face लॉगिन और मुख्य कोड रन करें
कोड चलाने से पहले लॉगिन करने के लिए यह कोड चलाएं और अपना कॉपी किया हुआ टोकन पेस्ट कर दें:
```python
from huggingface_hub import notebook_login
notebook_login()
```
इसके बाद मुख्य `main.py` वाला कोड रन करें। यह पहली बार में मॉडल डाउनलोड करेगा (लगभग 14.2 GB), जो कोलाब के सुपर-फास्ट इंटरनेट से 4-5 मिनट में हो जाता है।

### स्टेप 5: प्रॉम्प्ट डालें और मजे लें!
मॉडल लोड होते ही नीचे इनपुट बॉक्स खुलेगा। वहां अपना आइडिया इंग्लिश में टाइप करें और एंटर दबाएं। 
- आपकी फोटो सीधे स्क्रीन पर दिखेगी।
- फोटो आपके गूगल ड्राइव में `My Drive -> AI_Images` फोल्डर में सेव हो जाएगी।
- टूल को बंद करने के लिए बस इनपुट बॉक्स में **`exit`** टाइप करें।
